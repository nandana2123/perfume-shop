This is a full-stack web application for an online perfume shop built using the MERN stack (MongoDB, Express, React, Node.js). The platform allows users to browse and purchase perfumes, manage their profiles, and view their order history. The shop offers a simple and intuitive user experience, providing an easy way for customers to explore different fragrance options.
Features
User Authentication: Sign up, login, and logout functionality using JWT authentication.
Product Catalog: Display a list of perfumes, including details such as name, brand, and price.
Search & Filter: Allow users to filter perfumes based on categories, price range, and brands.
Shopping Cart: Add perfumes to a shopping cart and proceed to checkout.
Order Management: View order history, track status, and manage payment methods.
Admin Panel: Admins can add, update, and remove perfumes from the catalog.
Responsive Design: Optimized for mobile and desktop users.
